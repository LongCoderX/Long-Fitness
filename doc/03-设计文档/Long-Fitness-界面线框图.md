## 组件设计规范

### 按钮组件
```
主要按钮: [背景色: #FF6B35, 文字白色, 圆角8px]
次要按钮: [边框色: #FF6B35, 文字#FF6B35, 背景透明]
文字按钮: [文字色: #45B7D1, 无背景]
```

### 卡片组件
```
+----------------------------+
|                            |
|        卡片标题            |
|                            |
|  卡片内容区域，包含相关    |
|  信息和操作按钮           |
|                            |
+----------------------------+
```

### 表单组件
```
标签文字
[输入框 - 圆角4px, 边框#E2E8F0]
辅助文字 (可选)
```

### 导航组件
桌面端: 左侧固定导航栏，宽度240px
移动端: 底部标签栏，等高分布

## 交互状态说明

### 按钮状态
- 正常状态: 100% 透明度
- 悬停状态: 亮度提高10%
- 点击状态: 缩放95%
- 禁用状态: 50% 透明度

### 输入框状态
- 正常: 边框色 #E2E8F0
- 聚焦: 边框色 #45B7D1, 阴影效果
- 错误: 边框色 #F56565
- 成功: 边框色 #48BB78

### 加载状态
- 骨架屏动画
- 旋转加载指示器
- 进度条显示
